# 蛋鸡四大产品学习助手 🐔

一款专为蛋鸡产品知识学习设计的**PWA移动应用**，支持云端部署，可在手机上独立运行。

## 📱 应用特色

| 特性 | 说明 |
|------|------|
| **驾驶友好** | 语音播报功能，无需看屏幕 |
| **高效记忆** | 闪卡模式强化知识记忆 |
| **离线可用** | 安装后可离线使用 |
| **云端部署** | 支持多种云端部署方式 |
| **跨平台** | Android/iOS均可使用 |

## 🚀 功能模块

| 模块 | 图标 | 功能说明 |
|------|------|----------|
| **📚 总览** | 查看四大产品完整信息 |
| **🎴 闪卡** | 卡片式翻转学习 |
| **🔊 听学** | 语音学习模式（适合驾驶） |
| **📝 测验** | 知识检验测试 |
| **📊 进度** | 学习进度追踪 |

## 📲 手机安装指南

### 前提条件

应用已部署到云端服务器，获得访问地址（如 `https://your-domain.com`）

### 安装步骤

1. **在手机浏览器中打开**应用地址
2. **添加到主屏幕**：
   - **Chrome浏览器**：点击右上角 ⋮ → 添加到主屏幕
   - **Safari浏览器**：点击底部 ↑ → 添加到主屏幕
3. **完成安装**：应用图标将出现在手机主屏幕上

### 使用体验

- ✅ 像原生App一样打开
- ✅ 离线状态下也能使用
- ✅ 支持语音播报（驾驶时使用）

## ☁️ 云端部署

### 方式一：Docker部署（推荐）

```bash
# 1. 克隆项目
git clone <your-repo-url>
cd egg-products-learning-app

# 2. 启动应用
docker-compose up -d

# 3. 访问应用
http://your-server-ip:3000
```

### 方式二：Vercel/Netlify部署（免费）

1. 注册账号：https://vercel.com/ 或 https://www.netlify.com/
2. 连接GitHub仓库
3. 配置构建命令：`npm run build`
4. 配置输出目录：`dist`
5. 点击部署

### 方式三：手动部署

```bash
# 安装依赖
npm install

# 构建应用
npm run build

# 启动服务器
npm run serve
```
