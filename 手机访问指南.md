# 📱 手机访问指南

## ✅ 服务器已启动

**网络访问地址**：`http://172.16.0.101:3000`

---

## 📲 手机访问步骤

### 第一步：确保网络连接

✅ **确认手机和电脑连接到同一WiFi网络**

- 手机WiFi名称应与电脑WiFi名称相同
- 如果不确定，可以在电脑上运行：`ipconfig` 查看

---

### 第二步：在手机浏览器中打开

在手机浏览器（Chrome/Safari）中输入：

```
http://172.16.0.101:3000
```

**注意事项**：
- 不要输入 `localhost`，手机无法访问
- 确保输入完整的地址（包括 `http://`）
- 如果还是无法打开，尝试关闭手机浏览器的"省流模式"

---

### 第三步：添加到主屏幕

应用成功打开后，安装到手机桌面：

#### Chrome浏览器
1. 点击右上角 **⋮** 菜单
2. 选择 **"添加到主屏幕"**
3. 点击 **"添加"** 确认

#### Safari浏览器
1. 点击底部 **↑** 分享按钮
2. 选择 **"添加到主屏幕"**
3. 点击 **"添加"** 确认

---

## ❌ 如果还是无法打开

### 可能的原因和解决方案

| 问题 | 解决方案 |
|------|----------|
| **防火墙阻止** | 在Windows防火墙中允许3000端口 |
| **网络隔离** | 检查路由器是否启用了AP隔离 |
| **浏览器省流模式** | 关闭浏览器的省流/数据节省模式 |
| **IP地址变化** | 重新运行 `ipconfig` 获取新IP |

---

## 🔧 防火墙配置（如果需要）

### Windows防火墙设置

1. 打开 **控制面板** → **Windows Defender 防火墙**
2. 点击 **高级设置**
3. 选择 **入站规则** → **新建规则**
4. 选择 **端口** → **TCP** → **特定本地端口：3000**
5. 选择 **允许连接**
6. 应用到所有网络
7. 命名规则为 **"蛋鸡学习助手"**

---

## 🌐 测试网络连接

### 在手机上测试

1. 打开手机浏览器
2. 输入：`http://172.16.0.101:3000`
3. 如果显示应用界面，说明连接成功

### 如果无法连接

尝试ping测试（如果手机支持）：
```
ping 172.16.0.101
```
